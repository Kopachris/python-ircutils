IRCUtils 0.1.1


== Description ==
IRCUtils is a Python framework for communicating with the  IRC protocol. 
It's designed for ease of use for anything you'd need. It also has fast and 
simple support for writing bots.   


== Installing ==
   1. Extract the contents of ircutils-[v]-[v]-[v].tar.gz, where [v] are version
      numbers.
   2. Open up console and navigate to the file contents
   3. Run `python setup.py install`

 
 == Bug reporting ==
   1. Navigate to http://dev.guardedcode.com/projects/ircutils
   2. Click on "new ticket" and fill out the summary and description
   3. In the "type" field, specify "defect"
   4. Submit!
